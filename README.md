# The Scout — Football Analytics Intelligence Platform

A data-driven football scouting platform that processes 2,791 players across Europe's Big 5 leagues.

**Built with:** Python · Pandas · Plotly · Streamlit · football-data.org API · FBref

**Features:**
- Live squad data for 96 teams across Premier League, La Liga, Bundesliga, Serie A, Ligue 1
- Interactive formation display with player tier classification
- Position-based scout suggestions with 30+ player archetypes
- Team-specific colour theming with real club crests
